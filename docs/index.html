Scouting

Match Scouting: The 2024 Match Scouting Form is used to suppliment the Match Sim program. This a manual program, created by our programming team (me), to allow for match scouting so that people can have something to do at competitions, and allowing for a surplus of info to be collected. This form changes from year to year, but this form has questions such as: "How many points were scored in Auto", "Did the robot have any technical issues?", and other questions such as that. This was meant to provide more detailed information, and levels of information that simply simulating matches could not.

Team/Match Sim:
The Match Sim can be used to evaluate individual teams, alliances of 3 teams, or full "matches" of 3v3 teams. It takes data from the most recent season, and displays it in a variety of aggregated and non-aggregated forms.
For single team evalutations, it displays a middle number, known as the "Performance Number", surrounded by a ring, which can change from red, to green based on how well the team has performed. When clicked, the performance number/circle will toggle a more detailed menu, which includes stats such as win/loss ratios, average match scores ± standard deviation, and world/area rankings.
For full alliance evaluations, the tool will again return the performance circle, and the toggle will have all of the same data, excluding the rankings.
When a full "match" is entered, there will be no performance circle, but instead, there will simply be text that says "Point Diff" (Point Differential), accompanied by an amount of points, colored the color of the winning alliance. All "matches" for this function must be entered in the format of Red - Blue. When the "Point Diff" number is clicked on, a more detailed menu will expand. This menu contains estimated scores from Auto, Teleop and Endgame periods of the match, along with a graph representing each alliances scoring during these sections.
Input Formats (Match Sim): For inputs into the Match Sim program, the format is as follows. Single Team: #### (Note that ### teams are simply ###, not 0###). Alliance of teams: ####, ####, #### Full Match Sims: ####, ####, #### - ####, ####, ###
